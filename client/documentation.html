Get all (GET/HEAD)

Returns

JSON array of Pokemon objects

[{ id: number, num: string, name: string, img: string (url), type: [string], height: string, weight: string, weaknesses: [string], next_evolution: [{ num: string, name: string }, ] }]

Example

Request:

/getAll

Response:

[ { "id": 1, "num": "001", "name": "Bulbasaur", "img": "http://www.serebii.net/pokemongo/pokemon/001.png", "type": [ "Grass", "Poison" ], "height": "0.71 m", "weight": "6.9 kg", "weaknesses": [ "Fire", "Ice", "Flying", "Psychic" ], "next_evolution": [ { "num": "002", "name": "Ivysaur" }, { "num": "003", "name": "Venusaur" } ] }, ... ]

Search by Name (GET/HEAD)

Query Params

Name: String, pokemon name

Returns

[{ id: number, num: string, name: string, img: string (url), type: [string], height: string, weight: string, weaknesses: [string], next_evolution: [{ num: string, name: string }, ] }, ... ]

Example

Request:

/searchName?name=bulbasaur

Response:

[ { "id": 1, "num": "001", "name": "Bulbasaur", "img": "http://www.serebii.net/pokemongo/pokemon/001.png", "type": [ "Grass", "Poison" ], "height": "0.71 m", "weight": "6.9 kg", "weaknesses": [ "Fire", "Ice", "Flying", "Psychic" ], "next_evolution": [ { "num": "002", "name": "Ivysaur" }, { "num": "003", "name": "Venusaur" } ] }]

Search by Strong Against (GET/HEAD)

Gets Pokemon who are effective against the type entered

Query Params

effective: a pokemon type

Returns

[{ id: number, num: string, name: string, img: string (url), type: [string], height: string, weight: string, weaknesses: [string], next_evolution: [{ num: string, name: string }, ] }, ... ]

Example

Request:

/searchName?effective=Fire

Add Pokemon

Body Params

Name (optional)- New Pokemon's name.
Height (optional)- New height
weight (optional)- New weight
type1 (optional)- First new type
type2 (optional)- Second new type

Examples

Request /addPokemon
Body {name="newPoke", weight="8", type1="Water", type2="Ice"}
Result {"id":152,"num":"152","name":"newPoke","type":["Water","Ice"]}

Edit pokemon

Body Params

ID (optional)- ID of pokemon to edit
Name (optional)- if ID exists, new name for pokemon of [ID]. If ID not entered, pokemon to edit.
Height (optional)- New height
weight (optional)- New weight
type1 (optional)- First new type
type2 (optional)- Second new type

Examples

Request /editPokemon
Body {id=34, name="tehe", weight="24", type1=Fire}
Result { "id": 34, "num": "034", "name": "tehe", "img": "http://www.serebii.net/pokemongo/pokemon/034.png", "type": [ "Fire"], "height": "1.40 m", "weight": "24", "weaknesses": [ "Water", "Ice", "Ground", "Psychic" ] }
